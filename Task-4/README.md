# Tantis.pl autosuggest checker

> A simple example of using webdriver.io to check item from autosuggest on Tantis.pl shop.


## Build Setup

``` bash
# install dependencies
npm install
```

## Run test
### run test from root directory
``` bash
$ ./node_modules/.bin/wdio wdio.conf.js
```
